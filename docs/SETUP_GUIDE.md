# セットアップガイド

## 概要

このプロジェクトは `aicli-links.json` でリンク構成を管理しています。
リポジトリを clone したら、対話的セットアップツールでリンクを作成できます。

## 使い方

### 1. リポジトリをクローン

```bash
git clone <repository-url>
cd <repository-name>
```

### 2. セットアップを実行

```bash
aicli-config-bridge setup
```

### 3. 対話的に進める

セットアップツールが各リンクについて順番に確認します。

- ソースファイルが存在しない場合: デフォルト内容で作成するか選択
- ターゲットファイルが既に存在する場合: バックアップ、上書き、スキップから選択

### 4. 完了

すべてのリンクが正しく作成されます。

## トラブルシューティング

### 設計図が見つからない

```
❌ リンク設計図が見つかりません: aicli-links.json
```

プロジェクトルートで実行していることを確認してください。

### 既存ファイルとの衝突

既存ファイルがある場合は、バックアップを選択することを推奨します。
バックアップは `.aicli-backup/` ディレクトリに保存されます。
